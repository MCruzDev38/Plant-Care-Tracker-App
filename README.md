# Plant-Care-Tracker App

**Program Description:**
The Plant Care Tracker App is a Java console application that helps users manage plant care information. It allows users to add, view, update, and delete plant records. Each plant record stores the plant name, plant type, watering frequency, sunlight needs, and any additional notes the user would like to include, to a database that saves the plant records to help the user keep track of care requirements for multiple plants.

**Program Tools:**
- IntelliJ IDEA
- Java
- SQlite

**Program Concepts:**
- Object-oriented programming
- Classes and objects
- Constructors
- Getter methods
- User input with Scanner
- Switch statements
- Loops
- SQLite database connection
- SQL commands
- Prepared statements
- CRUD operations: Create, Read, Update, and Delete

**Expected Program Inputs:**
- Menu Selection (Add, View, Update, Delete, Exit)
- Plant Name
- Plant Type
- Watering Frequency
- Sunlight Requirements
- Plant Notes
Plant ID (for updates and deletions)

**Expected Program Outputs:**
- Confirmation messages when records are added, updated, or deleted
- A list of plant records stored in database
- Plant details that include plant ID for database tracking, name of plant, type, watering frequency, sunlight requirements, and notes
- Error messages for invalid input or missing records
